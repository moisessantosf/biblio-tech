# BiblioTech

Faça o clone deste projeto:

```
git clone git@github.com:moisessantosf/biblio-tech.git
```

Depois, acesse a pasta onde o projeto foi salvo (exemplo):

```
cd biblio-tech
```

Na pasta raíz do projeto, execute o comando através do Powershell:

```
docker-compose up --build
```

Para acessar a documentação da API (swagger), acesse em seu browser a URL http://localhost:8080/swagger/index.html .
